## Описание

Random Task Generator — это GUI-приложение на Python для генерации случайных задач.

Приложение использует библиотеку `random` и позволяет:

- выбирать случайную задачу из списка;
- фильтровать задачи по типу: учёба, спорт, работа;
- добавлять новые задачи;
- проверять корректность ввода при добавлении задачи;
- отображать историю сгенерированных задач;
- сохранять историю в JSON;
- загружать историю при повторном запуске программы.

## Как запустить проект

1. Установите Python 3.
2. Скачайте проект.
3. Откройте папку проекта в терминале.
4. Запустите команду:

```bash
python random_task_generator.py
```

Если команда не работает, попробуйте:

```bash
python3 random_task_generator.py
```

## Примеры использования

### Генерация случайной задачи

1. Выберите тип задачи: `Все`, `Учёба`, `Спорт` или `Работа`.
2. Нажмите кнопку `Сгенерировать задачу`.
3. Программа покажет случайную задачу и добавит её в историю.

### Добавление новой задачи

1. Введите текст задачи.
2. Выберите тип задачи.
3. Нажмите кнопку `Добавить`.

Если поле задачи пустое, программа покажет ошибку.

### Проверка сохранения истории

1. Сгенерируйте несколько задач.
2. Закройте приложение.
3. Откройте приложение снова.
4. История будет загружена из файла `task_history.json`.

## Возможные тесты

| Действие | Ожидаемый результат |
|---|---|
| Нажать `Сгенерировать задачу` | Появляется случайная задача |
| Выбрать фильтр `Спорт` | Генерируются только спортивные задачи |
| Добавить пустую задачу | Появляется сообщение об ошибке |
| Добавить корректную задачу | Задача сохраняется в `tasks.json` |
| Перезапустить приложение | История загружается из `task_history.json` |
